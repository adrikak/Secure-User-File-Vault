# Secure User File Vault (SUFV)

Secure User File Vault (SUFV) is a console-based file management system developed using C++, Object-Oriented Programming (OOP), and PostgreSQL. The project provides a secure environment for user authentication, file storage, and controlled access to user data through a database-driven architecture.

The system was designed to demonstrate the practical implementation of software engineering principles, database integration, and object-oriented design while providing a functional and user-friendly file management solution.

---

## Features

### User Authentication
- User Registration
- Secure Login System
- Password Verification
- User-specific Access Control

### File Management
- File Upload and Storage
- File Retrieval
- File Organization
- User-specific File Access

### Database Integration
- PostgreSQL Connectivity
- User Data Management
- Credential Storage and Verification

### Security Features
- Authentication Checks
- Access Restrictions
- Protected User Sessions
